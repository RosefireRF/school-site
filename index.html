HTML

HTML je opisni jezik specijalno namenjen opisu veb stranica. Pomoću njega se jednostavno mogu odvojiti elementi kao što su naslovi, paragrafi, citati i slično

CSS

CSS je jezik formatiranja pomoću kog se definiše izgled elemenata veb-stranice, u određenoj formi je postojao još u začecima SGML-a 1970ih godina.

JAVASCRIPT

Javaskript ili Džavaskript je dinamičan, slabo tipiziran i interpretiran programski jezik visokog nivoa.

SEO

SEO optimizacija veb-sajta za pretraživače je proces unapređenja kvaliteta i količine saobraćaja od pretraživača ka veb-sajtu preko prirodnih rezultata pretrage.
